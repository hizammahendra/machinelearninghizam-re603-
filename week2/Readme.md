# OOP Principles

Proyek ini merupakan implementasi konsep dasar Object Oriented Programming (OOP) menggunakan bahasa Python. Program memodelkan sebuah robot dengan atribut dan perilaku tertentu, serta menunjukkan bagaimana konsep OOP digunakan dalam pengembangan sistem yang terstruktur.

## Konsep OOP yang Digunakan

* Encapsulation: Atribut `speed` dibuat private dan hanya dapat diakses melalui method `get_speed()` dan `set_speed()`.
* Inheritance: Class `Smorphi` mewarisi atribut dan method dari class `Robot`.
* Object: Pembuatan objek `robot1` sebagai instance dari class `Smorphi`.

## Struktur Program

* `Robot` : Class dasar yang memiliki atribut nama dan kecepatan serta method untuk bergerak.
* `Smorphi` : Class turunan yang memiliki kemampuan tambahan yaitu transformasi bentuk.
* Main Program : Digunakan untuk membuat objek dan menjalankan method yang tersedia.


## Tujuan

* Memahami konsep dasar OOP
* Mengimplementasikan encapsulation dan inheritance
* Membuat struktur program yang lebih rapi dan modular

## Output Program

Program akan menampilkan:

* Status pergerakan robot
* Nilai kecepatan robot sebelum dan sesudah diubah
* Perubahan bentuk robot

## Catatan

Proyek ini dibuat sebagai latihan dasar untuk memahami penerapan OOP dalam Python.
