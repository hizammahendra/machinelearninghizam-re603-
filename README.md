# Introduction to Machine Learning

## Apa itu Machine Learning

Machine Learning adalah cabang dari kecerdasan buatan yang memungkinkan sistem komputer untuk belajar dari data tanpa perlu diprogram secara eksplisit. Model machine learning mengenali pola dalam data dan menggunakan pola tersebut untuk membuat prediksi atau keputusan.

## Jenis-Jenis Machine Learning

Secara umum, machine learning dibagi menjadi tiga kategori utama:

* **Supervised Learning**: Model dilatih menggunakan data berlabel (input dan output sudah diketahui).
* **Unsupervised Learning**: Model mencari pola dalam data tanpa label.
* **Reinforcement Learning**: Model belajar melalui interaksi dengan lingkungan menggunakan sistem reward dan punishment.

## Tujuan Machine Learning

Tujuan utama machine learning adalah membuat sistem yang dapat:

* Melakukan prediksi secara akurat
* Mengidentifikasi pola tersembunyi
* Membantu pengambilan keputusan berbasis data
